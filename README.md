# Kkutu-helper
끄투 온라인에서 사용하실 수 있는 단어장 헬퍼입니다.
(http://kkutu.co.kr/)

###사용법

Python 3 버전으로 제작되었습니다. (Window 에서 정상적인 실행 확인)


	C:\Users\rhdwk\Documents\Kkutu-helper>helper.py
	
	데이터를 성공적으로 불러왔습니다.
	
	원하는 단어를 얻기 위해서 필터를 설정해주세요!
	
	-----------------------------------
	
	1. 찾을 수 있는 단어 중 가장 긴 단어 찾기 / 받아치기 힘든 단어 순서대로 찾기
	
	2. 장르 필터 켜기
	
	
	0. 옵션 선택 종료
	
	-----------------------------------
	
	필터: 긴 단어 순서
	
	장르 필터: None!
	
	옵션을 선택해주세요!:

특정 장르에 있는 단어만 선택하여 필터링할 수 있으며,
긴 단어 순서 혹은 받아치기 힘든 단어 순서대로 가져올 수 있습니다.
